Math - about prime numbers

7 is a prime number
2^7-1=127
127 is a prime number
2^127-1 is also a prime number

127 is so special that it appears in pollard-rho algorithm, as a threshold value of iterating time

Prime number n is called Mersenne prime if 2^n-1 is also prime
19937 is also a Mersenne prime, which appears in Mt19937 (a random device, a function that can provide qulified random numbers)
